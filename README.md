# 🛒 UniMart – Hyper-Local Campus Marketplace

## 📌 Overview

**UniMart** is a hyper-local campus marketplace built using the **MERN stack (MongoDB, Express.js, React.js, Node.js)**.
It was inspired by a real problem on campus — lack of an efficient canteen system and long queues at printing shops.

UniMart aims to **digitize campus services** and allow students to order food, stationery, and printing services quickly instead of waiting 20–30 minutes.

---

## 🚀 Key Features

### 👨‍🎓 Students

* Order food and stationery items
* Upload PDFs for printing
* Track orders in real-time

### 🏪 Vendors

* Manage shop items through dashboard
* Accept and process food & print orders
* Update order status

### 🚴 Delivery Personnel

* Accept delivery requests
* Deliver orders across campus efficiently

---

## ⚡ Real-Time System

* Implemented using **Socket.io**
* Instant updates when:

  * Vendor completes an order
  * Delivery personnel receive notifications
* Reduces communication delay and improves efficiency

---

## 🔐 Authentication & Security

* **JWT Authentication**
* **Google OAuth Login**
* **OTP-based Verification System**

  * Ensures correct delivery to students

---

## 📂 File Uploads

* **Multer** for handling file uploads
* **Cloudinary** for storing:

  * PDF files (printing)
  * Product images

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Redux (State Management)

### Backend

* Node.js
* Express.js

### Database

* MongoDB (with structured schemas for users, shops, and orders)

---

## 🎯 Project Goal

The main goal of UniMart was to build a **functional MVP (Minimum Viable Product)** that demonstrates how a campus ecosystem can be:

* Digitized
* Organized
* Made more efficient

---

## 💡 Problem Solved

* ❌ Long queues in canteens & printing shops
* ❌ No centralized ordering system
* ❌ Time wastage for students

✅ **Solution:** One platform for ordering, tracking, and delivery
